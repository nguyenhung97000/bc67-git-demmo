git : (phần mềm quản lí code) version control system

git hub : nơi lưu trữ code online ( icloud, drive, ...)

git init : khởi tạo repo local

git remote add origin https://github.com/NguyenHung0908/bc67-demo-git.git : liên kết repo local và repo remote

đưa code lên github gồm 3 bước 
git add -add
git commit -m "nội dung commit"
git push -u origin master ( lần đầu tiên ) / git push ( lần thứ 2 trở đi )


tạo nhánh / chuyển nhánh : commit trước khi thao tác

git branch : hiển thị nhánh hiện tại
git cheackout tenNhanhCanChuyen : chuyển sang nhánh khác
